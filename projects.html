PacMan

This repository offers a modern rendition of the classic Pac-Man game, crafted with finesse using JavaScript and the powerful p5.js library.

Seamlessly integrated into an HTML framework, players can enjoy the game on any web browser. The meticulous 2D grid representation of the game arena and the iconic A* path-finding technique drive the ghost movement.

Experience the essence of Pac-Man, reimagined for today's web-centric world.

Sudoku

This repository presents an engaging Sudoku game, crafted with JavaScript and the p5.js library, providing enthusiasts with a digital platform to tackle various puzzles.

Players interact with a standard 9x9 Sudoku grid on an HTML canvas. The game offers a plethora of puzzles, navigable via dedicated buttons, ensuring a fresh experience with every switch.

Whether you're testing your puzzle-solving skills or seeking guidance on a tricky grid, this game has got you covered.

Rock Paper Scissor Simulator

This Rock-Paper-Scissors simulation project, implemented using p5.js, creates a dynamic environment where entities interact based on classic game rules.

The entities move across the canvas, engaging with one another, consuming and reproducing under certain conditions, all governed by configurable parameters.

Enhanced with preloaded images and interactive features, this simulation offers a fascinating exploration of a traditional game in a digital context.
